# 피켓 PDF 업로드 위치

`/fair` 페이지의 "피켓 다운로드" 버튼이 가리키는 정적 파일 경로입니다.

## 필요한 파일

| 경로 | 문구 |
|------|------|
| `picket-01.pdf` | 자료 공개 없는 검증은 검증이 아닙니다 |
| `picket-02.pdf` | 셀프 감사 말고 문서를 공개하라 |
| `picket-03.pdf` | 그 연구실 문, 우리 아이에게도 열립니까 |
| `picket-04.pdf` | 국가 연구성과가 왜 후보 아들 스펙이 됐습니까 |
| `picket-05.pdf` | 권순기는 서울대 입학 서류를 공개하라 |
| `picket-06.pdf` | 권순기는 연구기여도 자료를 공개하라 |

## 규격

- A4 출력용 (인쇄 후 25cm 이내로 절단)
- 흑백 인쇄 가독성 필수 (큰 글씨)
- 절단선 포함 권장

## 업로드

이 디렉토리에 동일 파일명으로 PDF를 두면 빌드 없이 다음 배포부터 바로 다운로드됩니다.
git에 커밋 → push → Cloudflare Pages 자동 배포.
